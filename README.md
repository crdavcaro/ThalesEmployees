# Thales Employees

Project to apply for the Senior .NET Developer position.

## Technologies

The technologies and libraries used in the project are the following:

- .NET Core
  - Auto Mapper
  - MsTest
- Angular 16
  - PrimeNg
  - Prime Flex
  - Prime Icons
 
## How to Start

- Restore NuGet packages
```bash
nuget restore
```

- Restore npm Packages
```bash
  npm install
```
